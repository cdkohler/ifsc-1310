Thoughts on Progressive Enhancement

What is Progressive Enhancment? My definition on Progressive Enhancement is that it is a way to code a document, page, or website in layers. It is how we have coded this semester to create this homepage. How ProgressiveEnhancement starts is with the HTML, then it moves to the CSS, and finialy the JAVA script. when you have all these "layers" done you have a finshed document or website modeled withe the style of Progressive Enhancement.

Progressive Enhancement is important in modern web design because it is very simple and allows for everyone to add on to a page if they choose to. this allows for the people building the website or page to make the best possible they can, and to do so very easily. So a short description of how Progressive Enhancment is important is that is is a simple way of coding that provides the become possible outcome for a webiste.
